# 📖 Framework de Análise SRL — Visão Geral

## 🚀 Fluxo do Framework

1. **MinimalDataset** → preparação do dataset mínimo a partir dos eventos brutos da plataforma Moodle  
2. **ExploratoryAnalysis** → análise exploratória, estatísticas e correlações  
3. **Clustering** → execução de algoritmos de clusterização (KMeans, DBSCAN, HDBSCAN, etc.)  
4. **ResultsAnalysis** → avaliação dos clusters, testes de significância e descrição de perfis SRL  
5. **General** → orquestrador que integra a etapa de análise exploratória, clustering e análise de resultados a partir de um conjunto de dados no formato do minimal dataset
