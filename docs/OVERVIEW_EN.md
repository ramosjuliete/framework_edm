# 📖 SRL Analysis Framework — Overview

## 🚀 Framework Flow
1. **MinimalDataset** → preparation of the minimal dataset from raw Moodle platform events  
2. **ExploratoryAnalysis** → exploratory analysis, statistics, and correlations  
3. **Clustering** → execution of clustering algorithms (KMeans, DBSCAN, HDBSCAN, etc.)  
4. **ResultsAnalysis** → evaluation of clusters, significance tests, and SRL profile descriptions  
5. **General** → orchestrator that integrates exploratory analysis, clustering, and results analysis from a dataset in minimal dataset format
